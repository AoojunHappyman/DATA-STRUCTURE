# 📚 Data Structure Repository

โปรเจกต์นี้เป็นการรวบรวมโค้ดและสรุปเนื้อหาเกี่ยวกับ **Data Structure**
เพื่อใช้ในการเรียน ฝึกทำโจทย์ และทบทวนก่อนสอบ

---

## 🔹 1. Array-Based Sequences

โครงสร้างข้อมูลพื้นฐานที่ใช้เก็บข้อมูลแบบเรียงลำดับในหน่วยความจำต่อเนื่อง

### 📌 หัวข้อสำคัญ

* Array / List
* Dynamic Array
* การเพิ่มและลบข้อมูล (Insert / Delete)
* Time Complexity
* การเข้าถึงข้อมูลด้วย Index

---

## 🔹 2. Recursion

แนวคิดการแก้ปัญหาโดยให้ฟังก์ชันเรียกตัวเอง

### 📌 หัวข้อสำคัญ

* Base Case / Recursive Case
* Factorial
* Fibonacci
* Tree Recursion
* แนวคิด Backtracking เบื้องต้น

---

## 🔹 3. Trees & Search Trees

โครงสร้างข้อมูลแบบต้นไม้ ใช้ในการจัดการข้อมูลแบบลำดับชั้น

### 📌 หัวข้อสำคัญ

* Binary Tree
* Binary Search Tree (BST)
* Tree Traversal:

  * Inorder
  * Preorder
  * Postorder

---

## 🔹 4. Hashing & Hash Table

โครงสร้างข้อมูลที่ใช้ Hash Function ในการจัดเก็บและค้นหาข้อมูลอย่างรวดเร็ว

### 📌 หัวข้อสำคัญ

* Hash Function
* Collision
* Chaining
* Open Addressing
